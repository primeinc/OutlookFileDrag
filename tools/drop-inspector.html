Drag/Drop Inspector

Drag an Outlook email or attachment onto the box. It probes every channel a web drop handler can reach and reads the actual bytes. (You can also paste with Ctrl+V.)

⬇️  Drop an Outlook item (or any file) here

Test matrix: (1) classic Outlook with the add-in active → expect real files. (2) classic Outlook with the add-in disabled (HKCU\Software\OutlookFileDrag\EnableHook = 0, restart Outlook) → the key test. (3) New Outlook. Watch whether files is empty but items[].getAsFile / webkitGetAsEntry / getAsFileSystemHandle still deliver real bytes. The status bar turns green if real file content reaches JavaScript.
